# UI Reference

This directory stores visual reference material for dashboard redesign work.

## Files

- `cosmic-gravity-ui-reference.jpeg`: the cosmic-black, black-gold dashboard reference used for the current frontend redesign.
- `lightcore-prism-ui-reference.jpg`: a bright prism-style UI design system reference for possible light-mode or alternate visual exploration.
- `dashboard-redesign-live-complete.png`: core page mockup for the full `实时监控室` implementation.
- `dashboard-redesign-review-overview.png`: core page mockup for `复盘室 / 总体复盘`.
- `dashboard-redesign-daily-review.png`: core page mockup for `复盘室 / 每日复盘`.
- `dashboard-redesign-system-status.png`: core page mockup for `系统状态室`.

## Supporting Drafts

- `dashboard-redesign-live-first-screen.png`: earlier live-room first-screen draft. It is superseded by `dashboard-redesign-live-complete.png` and is not counted as a core page mockup.

## Notes

- `cosmic-gravity-design-notes.md`: notes for the Cosmic Gravity dark dashboard direction.
- `lightcore-prism-design-notes.md`: notes for the Lightcore Prism bright UI direction.

## Usage

- Keep reference images here, not in code or docs directories.
- Use these assets as design targets when adjusting layout, spacing, typography, color, and component styling.
