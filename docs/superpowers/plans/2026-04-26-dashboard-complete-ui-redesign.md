# Dashboard Complete UI Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Update the server-rendered Momentum Alpha dashboard to match the documented complete UI redesign while preserving current dashboard data, routing, and trading behavior.

**Architecture:** Keep the existing Python server-rendered dashboard. Recompose room-level HTML in the existing `dashboard_render_*` modules, keep data assembly in `dashboard_render_shell.py`, and implement the visual system through the existing dashboard asset style modules.

**Tech Stack:** Python 3, `unittest`, server-rendered HTML strings, inline CSS generated by `dashboard_assets_styles_*`, existing dashboard JavaScript.
